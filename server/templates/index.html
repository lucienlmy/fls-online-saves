How It Works

FSL downloads your actual online save file when no local saves are found. All subsequent save and load requests are redirected to your local hard drive. Your save data isn't stored on Rockstar's servers anymore.

All save data files are stored in %APPDATA%\FSL. The tool creates backups regularly, but you should maintain your own backups in a safe location.

FAQ

Is this safe?

This should be nearly undetectable since your saves never reach Rockstar's servers. It is recommended to use a menu with a working anti-cheat bypass when going online.

Can I edit my saves?

There are no tools to edit save files yet. You can use CodeWalker to view them.

Can I share my save files?

Yes, you can share save files with other people.
